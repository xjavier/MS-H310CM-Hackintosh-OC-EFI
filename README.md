# MS-H310CM-Hackintosh-OC-EFI
MAXSUN铭瑄挑战者-H310CM-i5-8500-RX580

适用于8/9代coffeelake平台

基于 OpenCore 0.6.0 版本，机型 iMac19,1，系统 macOS 10.15.7，实测升级OC版本可以完美使用macOS11、12、13系列

| 内存        | 协德  2666MHz 16G * 2 |
